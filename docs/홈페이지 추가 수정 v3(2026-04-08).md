### ✅ 6. 무료 사업지 검토 신청 이메일 전송
**파일:** `app/api/contact/route.ts`, `.env.local`  
**현황:** API 구현 완료, SMTP 설정 필요  
**수정:** `.env.local`에 Gmail 앱 비밀번호(`SMTP_PASS`) 추가  
**수신 이메일:** `pinp10000@gmail.com`

#### Gmail 앱 비밀번호 설정 방법
1. Google 계정 → 보안 → 2단계 인증 활성화
2. Google 계정 → 보안 → 앱 비밀번호 생성
3. `.env.local`에 추가:
   ```
   SMTP_PASS=발급받은_앱비밀번호
   ```
