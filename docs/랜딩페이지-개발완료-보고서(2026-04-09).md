## 3. 주요 기능

### 3-1. 네비게이션
- 히어로 구간(600px 이하): 투명 배경 + 흰 텍스트
- 히어로 이후: 흰 배경 + 어두운 텍스트
- 클릭 시 해당 섹션으로 부드럽게 스크롤
- 활성 섹션 하단 실선 애니메이션

### 3-2. 카운터 애니메이션 (About 섹션)
- **16** 단지 / **6,182** 세대 / **60+** 컨설팅
- easeOutQuart 커브, 2.4초, 60fps
- 화면에 진입 시 1회 실행 (IntersectionObserver)

### 3-3. 포트폴리오 캐러셀 (Projects 섹션)
- 16개 프로젝트 카드 가로 스크롤
- 좌/우 화살표 네비게이션
- CSS background-image 방식으로 이미지 렌더링

### 3-4. 상담 신청 폼 (Contact 섹션)
- 성함, 연락처, 문의 유형, 상세 내용
- API Route: `/api/contact` (Google Sheets 또는 이메일 전송)
- 제출 완료 시 성공 메시지 표시 (6초 후 초기화)

### 3-5. 비즈니스 파트너 (Footer)
- GS건설 / 쌍용건설 / 대우건설 / SK에코플랜트 / 한화건설 / DL이앤씨
- 컬러 코딩된 로고 박스 (이미지 불필요)

## 4. 공통 아키텍처

### 공통 Hook
```
hooks/useReveal.ts  ← IntersectionObserver 공통화
```
- 6개 컴포넌트(About, Services, Solution, Contact, Projects, NewsInsight)에서 공유
- `.reveal` 클래스 요소가 뷰포트 진입 시 `.visible` 클래스 추가 → CSS 애니메이션

### 이미지 파일 구조
```
public/image/
├── hero_bg_grey.png       ← Hero 배경
├── logo_transparent.png   ← 로고
├── ceo_photo.jpg          ← 대표이사 사진
├── success_handshake.jpg  ← Contact 섹션 이미지
├── about_expertise.png    ← About 카드 이미지
├── about_trust.png
├── about_continuity.png
├── milestone_2013.png     ← 연혁 이미지
├── milestone_2026.png
├── portfolio_01~16.png    ← 포트폴리오 더미 이미지
└── partner_*.svg/png      ← 파트너 로고
```
