# 랜딩페이지 검증 스킬 - 최종 사용 가이드

## 📋 검증 스킬 개요

**목적**: 랜딩페이지 완성 후 전체 품질 검증  
**형식**: 간단 리포트 (체크리스트 + 자동 스크립트)  
**범위**: 디자인/성능/SEO/UX + 작동 확인/오류 검사

---

## 🚀 사용 방법

### Step 1: 체크리스트 작성
```bash
# 파일 다운로드
landing-page-validation-checklist.md

# 4개 카테고리 + 작동 확인 섹션에서 체크박스 완성
```

### Step 2: 자동 검증 실행
```bash
python landing-page-check.py your-landing.html
```

**출력 결과:**
- ✅ 정보 (ℹ️): 진행상황, 참고 항목
- ⚠️ 경고 (경고): 개선 권장
- ❌ 오류 (실패): 필수 수정

### Step 3: 리포트 평가
```
결과: 오류 0개 | 경고 1개 | 정보 18개
└─> ⚠️ 개선 권장 후 배포
```

## 🔍 검증 항목 상세

### A. 기본 검증 (체크리스트)

| 카테고리 | 항목 수 | 점수 | 확인 내용 |
|---------|--------|------|---------|
| 📱 디자인/UI | 10개 | 30점 | 반응형, 색상, 레이아웃 |
| ⚡ 성능/속도 | 10개 | 20점 | 로딩, 이미지, 캐싱 |
| 🔍 SEO/최적화 | 13개 | 25점 | 메타, 구조화 데이터 |
| 👥 UX/기능성 | 15개 | 25점 | 버튼, 링크, 폼 |
| **소계** | **48개** | **100점** | |

### B. 작동 확인 & 오류 검사 (자동)

| 항목 | 검사 내용 | 자동화 | 수동 |
|------|---------|--------|------|
| 🔗 링크 작동 | URL 유효성, 404 감지 | ✓ | □ |
| 🔘 버튼 작동 | CTA, 클릭 이벤트 | ✓ | □ |
| 📋 폼 검증 | 입력 필드, 필수값 | ✓ | □ |
| 🖥️ 콘솔 오류 | JS/CSS 경로, undefined | ✓ | □ |
| ⚡ 성능 오류 | 파일 크기, 리소스 | ✓ | □ |
| 📱 모바일 호환성 | viewport, 반응형 | ✓ | □ |
| 🔒 보안 검사 | Mixed Content, API 키 | ✓ | □ |
| ♿ 접근성 | H1, alt 텍스트 | ✓ | □ |

---

## 📊 평가 기준

### 점수 계산 (100점 기준)

```
정상: 오류 0개, 경고 0개
→ ✅ 배포 가능

주의: 오류 0개, 경고 1-3개
→ ⚠️ 개선 권장 후 배포

실패: 오류 1개 이상
→ ❌ 필수 수정 필요
```

### 오류 심각도

| 심각도 | 예시 | 조치 |
|--------|------|------|
| 🔴 Critical | 링크/버튼 미작동, Mixed Content | 즉시 수정 |
| 🟠 High | 폼 검증 실패, 이미지 로드 오류 | 배포 전 수정 |
| 🟡 Medium | 성능 느림, alt 텍스트 부족 | 배포 후 개선 |
| 🟢 Low | 코드 최적화, 주석 추가 | 선택 사항 |

## 🛠️ 자동 검증 스크립트 활용

### 1. 링크 검사
```
[LINKS]
  총 링크: 4개
  앵커 링크: 4개
  외부 링크: 0개
  ✓ 깨진 링크 없음
```

### 2. 버튼/폼 검사
```
[BUTTONS]
  총 버튼: 0개
  CTA 버튼: 0개

[FORMS]
  폼 개수: 0개
  입력 필드: 0개
```

### 3. 오류 감지
```
❌ 오류 (0)
⚠️ 경고 (1)
  [BUTTONS] ⚠ 버튼 없음 (상호작용 요소 필요)
ℹ️ 정보 (18)
```

## 📝 검증 체크리스트 구성

```markdown
1. 프로젝트 정보 (이름, 날짜, 검증자)
2. 디자인/UI (10개 항목, 30점)
3. 성능/속도 (10개 항목, 20점)
4. SEO/최적화 (13개 항목, 25점)
5. UX/기능성 (15개 항목, 25점)
6. 점수 계산 (총 100점)
7. ⭐ 주요 문제점 (높음 우선순위)
8. ⭐ 작동 확인 & 오류 검사
   - 링크 작동 확인
   - 버튼 작동 확인
   - 폼 검증
   - 콘솔 오류
   - 성능 오류
   - 모바일 호환성
   - 브라우저 호환성
   - 보안 오류
   - 성능 타이밍
   - 오류 요약
9. 개선사항 (중간/낮음 우선순위)
10. 최종 승인 (완료/배포/재검증)
```

## 🔧 문제 해결

### 스크립트 실행 오류
```bash
# Python 설치 확인
python --version

# 파일 인코딩 확인 (UTF-8)
file -i test-landing.html

# 권한 설정
chmod +x landing-page-check.py
```

### 링크 검사 실패
```
원인: 절대 경로 vs 상대 경로 혼용
해결: 모든 링크를 /로 시작하는 절대 경로로 통일
```

### 콘솔 오류 감지 안 됨
```
원인: 자동 스크립트는 정적 분석만 가능
해결: 브라우저 DevTools > Console에서 실시간 확인
```
