## 디렉토리 구조

```
homepage/
├── app/
│   ├── page.tsx              ← 메인 페이지 (컴포넌트 조합)
│   ├── layout.tsx            ← 전체 레이아웃, 폰트 설정
│   └── api/contact/route.ts  ← 상담 폼 API (이메일/Sheets 전송)
├── components/
│   ├── Navigation.tsx        ← 고정 헤더, 스크롤 반응
│   ├── Hero.tsx              ← 메인 배너, 핵심 통계
│   ├── About.tsx             ← 회사 소개, CEO, 연혁, 카운터
│   ├── NewsInsight.tsx       ← 부동산 뉴스 캐러셀
│   ├── Services.tsx          ← 3대 서비스 카드
│   ├── Projects.tsx          ← 성공 프로젝트 + 포트폴리오 캐러셀
│   ├── Solution.tsx          ← 3가지 문제 해답
│   ├── Contact.tsx           ← 상담 신청 폼
│   └── Footer.tsx            ← 파트너 로고, 회사 정보
├── hooks/
│   └── useReveal.ts          ← 공통 IntersectionObserver hook
├── public/image/             ← 이미지 파일 (PNG, JPG, SVG)
├── docs/                     ← 개발 문서
├── .env.local                ← 환경변수 (git 제외)
├── next.config.js
├── tailwind.config.ts
└── package.json
```

## 개발 서버 실행

```bash
cd C:\Users\청라V1부동산\homepage
npm run dev
# → http://localhost:3000 자동 실행
```

> 포트 충돌 시 3001, 3002, 3003... 순서로 자동 변경됨

## 로컬 서버 다운 시 복구

```bash
# 기존 Node 프로세스 정리 후 .next 캐시 삭제
rm -rf .next
npm run dev
```

## 빌드 검증 (배포 전 필수)

```bash
npm run build
# ✓ Compiled successfully 확인 후 push
```

## 패키지 설치

```bash
npm install          # 의존성 전체 설치
npm install [패키지] # 새 패키지 추가
```

## 주요 npm 명령어

```bash
npm run dev     # 개발 서버 (localhost:3000)
npm run build   # 프로덕션 빌드 (배포 전 검증용)
npm run start   # 빌드된 앱 실행
npm run lint    # 코드 품질 검사
```
