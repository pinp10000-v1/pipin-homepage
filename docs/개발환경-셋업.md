# 개발 환경 셋업 가이드

## 기술 스택

| 항목 | 버전 |
|------|------|
| Node.js | 18.17+ |
| Next.js | 15 |
| React | 19 |
| TypeScript | 5.x |
| Tailwind CSS | 4.x |

## 주요 디렉토리 설명

### app/
- `page.tsx`: 메인 페이지
- `layout.tsx`: 전체 레이아웃 (헤더, 푸터)
- `api/contact/route.ts`: 문의 이메일 API

### components/
- `Navigation.tsx`: 상단 메뉴
- `Hero.tsx`: 메인 배너
- `About.tsx`: 회사 소개
- `Solution.tsx`: 솔루션
- `Projects.tsx`: 프로젝트/실적
- `Contact.tsx`: 문의 폼
- `Footer.tsx`: 하단 푸터

### public/image/
이미지 파일들 (PNG, JPG 등)

## 개발 서버 실행

```bash
npm run dev
```
- 자동으로 http://localhost:3000 열림
- 파일 수정하면 자동 새로고침

## 빌드 및 배포

```bash
# 빌드 확인
npm run build

# 빌드된 버전 실행
npm run start
```

## 패키지 설치

```bash
npm install  # package.json의 의존성 설치
npm install [패키지명]  # 새 패키지 추가
```

## 유용한 npm 명령어

```bash
npm run dev       # 개발 서버
npm run build     # 프로덕션 빌드
npm run start     # 빌드된 앱 실행
npm run lint      # 코드 검사
```
