```tsx
// ✅ 함수형 컴포넌트 + 'use client' (상호작용 있을 때)
```

## 이미지 처리 규칙

현재 프로젝트는 `next.config.js`에서 `unoptimized: true` 설정.  
이미지는 **`<img>` 태그** 또는 **CSS background-image** 방식 사용:

```tsx
// ✅ img 태그 방식 (파트너 로고, 성공 사례 이미지 등)
{/* eslint-disable-next-line @next/next/no-img-element */}
<img src="/image/success_handshake.jpg" alt="파트너십" style={{ width: '100%', height: '100%', objectFit: 'cover' }} />

// ✅ CSS background-image 방식 (포트폴리오 카드)
<div style={{ backgroundImage: `url(${project.image})`, backgroundSize: 'cover' }} />

// ✅ Next.js Image (로고, CEO 사진 등 최적화 필요한 경우)
import Image from 'next/image'
<Image src="/image/logo_transparent.png" alt="로고" fill className="object-contain" />
```


## Tailwind CSS 규칙

```tsx
// ✅ Tailwind 유틸리티 클래스 사용
<div className="flex items-center justify-between p-8 bg-white border border-gray-100">

// ✅ 반응형: 모바일 퍼스트
<h2 className="text-3xl md:text-5xl font-black text-navy">

// ✅ 인라인 스타일은 Tailwind로 표현 불가한 경우만 허용
<div style={{ backgroundImage: `url(${img})` }} />

// ❌ 불필요한 인라인 스타일
<div style={{ display: 'flex', color: '#333' }} />
```

|------|------|------|
| 컴포넌트 | PascalCase | `Navigation.tsx`, `Projects.tsx` |

| 폴더 | lowercase | `components/`, `hooks/` |
