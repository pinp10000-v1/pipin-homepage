# 피플인피플 홈페이지 - 개발 가이드

## 🎯 핵심 원칙

- **답변 & 주석**: 한국어로만 작성
- **가상환경**: 기존 `.venv` 있으면 재사용, `uv` 사용
- **시각화**: matplotlib은 `koreanize-matplotlib` 사용, seaborn 스타일 미사용

---

## ⚡ 배포 (3줄)

```bash
git add .
git commit -m "수정 내용"
git push origin master
```
→ 1~2분 후 https://www.pip-biz.com 자동 반영

---

## 📁 프로젝트 구조

```
homepage/
├── app/              # Next.js 페이지 & API
├── components/       # React 컴포넌트 (Navigation, Hero, About 등)
├── public/image/     # 이미지
├── docs/             # 문서
└── .env.local        # 환경변수 (GitHub 제외)
```

---

## 🚀 개발 환경

- **기술**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **시작**: `npm run dev` → http://localhost:3000

---

## 📋 자주 하는 작업

| 작업 | 방법 |
|------|------|
| **텍스트 수정** | `components/` 파일 수정 → `git push` |
| **이미지 교체** | `public/image/` 폴더에 동일한 파일명으로 저장 |
| **배포 실패** | Vercel 대시보드 → 빨간색 로그 확인 |
| **이메일 문의** | `.env.local`의 SMTP 설정 확인 |
